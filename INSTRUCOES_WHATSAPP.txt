3. CONTROLE DE PERMISSÕES
==========================

O sistema utiliza o sistema de autenticação do Django com verificação adicional:

- Apenas usuários com is_staff=True ou is_superuser=True podem acessar
- Usuários comuns são bloqueados e redirecionados ao site público
- Cada rota administrativa é protegida pelo decorator @admin_required

Código de verificação (decorators.py):
```python
def admin_required(view_func):
    @wraps(view_func)
    def wrapper(request, *args, **kwargs):
        if not request.user.is_authenticated:
            return redirect('admin_panel:login')
        if not (request.user.is_staff or request.user.is_superuser):
            return redirect('pages:home')
        return view_func(request, *args, **kwargs)
    return wrapper
```

5. ROTAS ADMINISTRATIVAS
=========================

/admin-panel/              - Dashboard
/admin-panel/login/        - Login
/admin-panel/logout/       - Logout

/admin-panel/services/           - Listar serviços
/admin-panel/services/create/    - Criar serviço
/admin-panel/services/<id>/edit/ - Editar serviço
/admin-panel/services/<id>/delete/ - Excluir serviço
/admin-panel/services/<id>/toggle/ - Ativar/desativar

/admin-panel/testimonials/           - Listar depoimentos
/admin-panel/testimonials/<id>/      - Ver depoimento
/admin-panel/testimonials/<id>/approve/ - Aprovar
/admin-panel/testimonials/<id>/reject/  - Reprovar
/admin-panel/testimonials/<id>/delete/  - Excluir
